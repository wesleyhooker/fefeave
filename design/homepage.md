# Fefe Ave — Homepage

Layout and content structure for the public homepage, as reflected in the final mockup. Use this as the single reference for section order and key elements.

---

## Section order

1. **Hero**
2. **Trust bar**
3. **How it works**
4. **Meet Felicia**
5. **Footer**

---

## Hero

- **Headline (H1):** “Catch the next drop!”
- **Subheadline:** “Join Felicia for live clothing drops and curated finds.”
- **Primary CTA:** Button (gold) — e.g. “Join the next live”
- **Secondary CTA:** Button (white, stone border) — e.g. “Follow on Whatnot”
- **Visual:** Hero image (Felicia / boutique) supporting the warm, inviting direction. Bright, natural, soft.

**Intention:** One clear value statement and two clear actions. Hero sets the tone for the rest of the page.

---

## Trust bar

- **Placement:** Directly below hero; full-width or within container.
- **Content:** Short, scannable stats or claims with small icons (e.g. “200+ live sales,” “1,500+ items sold,” “Trusted by resellers”).
- **Style:** Neutral background (e.g. Warm Sand); icons can use Soft Gold. Inter for text.

---

## How it works

- **Purpose:** Explain the value in a few steps (e.g. join live shows, discover deals, get paid clearly).
- **Layout:** Cards or rows; each step has icon, title, short description, optional “View schedule” or similar link.
- **Optional:** “View full schedule” or primary CTA below the cards.
- **Clarity:** If multiple blocks share the same structure, use distinct headings per block (e.g. one for “Explore live shows,” one for “Understand your earnings”) so each block has a clear purpose.

---

## Meet Felicia

- **Headline (H2):** “Meet Felicia” (or similar).
- **Body:** Short, warm copy about Felicia and the brand — host, boutique, community.
- **Link:** “Read our story” (tertiary/link style).
- **Image:** Friendly, approachable photo of Felicia (candid, natural light, warm).

---

## Footer

- **Content:** Minimal. E.g. “Follow on Whatnot,” “View upcoming shows,” and/or repeat primary CTA.
- **Optional:** Light nav links, social icons, legal/copyright. Keep visual weight low.

---

## Summary

| Section      | Key elements                                            |
| ------------ | ------------------------------------------------------- |
| Hero         | H1, subheadline, primary CTA, secondary CTA, hero image |
| Trust bar    | Stats/claims + icons, neutral bar                       |
| How it works | Cards/steps, icons, optional “View full schedule”       |
| Meet Felicia | H2, body copy, “Read our story,” photo                  |
| Footer       | CTAs and/or light nav, minimal                          |

This structure will drive implementation; components and spacing follow `components.md` and `spacing.md`.
